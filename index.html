Skills

HTML

HTML is the standard markup language for documents designed to be displayed in a web browser. It defines the structure of a web page. Think of it as the skeleton of a building. It uses "tags" (like h1, p, div) to tell the browser what is a heading, what is a paragraph, and where to put images.

CSS

CSS is a style sheet language used for describing the presentation of a document written in HTML. If HTML is the skeleton, CSS is the skin, clothes, and paint. It controls colors, fonts, spacing, and how elements are positioned on the screen. It allows you to make your website look beautiful and responsive on different devices.

JavaScript

JavaScript is a programming language that allows you to create interactive and dynamic content. While HTML and CSS give a page structure and style, JavaScript gives it "behavior." It handles things like button clicks, animations, data updates without refreshing the page, and complex logic.

Python

Python is a high-level programming language known for its simplicity and readability. It is used in web development, data analysis, artificial intelligence, automation, and game development. Python is popular among beginners and professionals because of its clear syntax.

Git

Git is a version control system that helps developers track changes in code. It allows working in teams, saving project history, and returning to previous versions if something goes wrong. Git is commonly used with platforms like GitHub and GitLab.
